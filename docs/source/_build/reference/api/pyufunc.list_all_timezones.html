pyufunc.list_all_timezones#

pyufunc.list_all_timezones(region_name='*')#

List all available timezones

Returns:

a set of all available timezones

Return type:

set

Example

>>> from pyufunc import list_all_timezones
>>> list_all_timezones()
{'Africa/Abidjan',
'Africa/Accra',
'Africa/Addis_Ababa',
'Africa/Algiers',
'Africa/Asmara',
'Africa/Asmera',
'Africa/Bamako',
'Africa/Bangui',
...}

>>> list_all_timezones("Chicago")
>>> ['America/Chicago']
