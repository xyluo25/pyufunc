Package overview#

pyufunc is a Python package providing fast, flexible, and expressive utility functions. It aims to be the fundamental high-level building block for doing practical, real-world utility functions in Python.

Getting support#

The first stop for pyufunc issues and ideas is the GitHub Issue Tracker.

Development team#

The list of the Core Team members and more detailed information can be found on the pyufunc website.

Institutional partners#

The information about current institutional partners can be found on pyufunc website page.
